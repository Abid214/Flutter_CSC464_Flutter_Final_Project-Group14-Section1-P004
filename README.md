# Expense Calculator

A simple Flutter app for managing daily expenses with Firebase integration.

## Features

- Add, edit, delete expenses
- View expense list with search, filter, and sort
- Analytics with total and category-wise expenses
- Real-time sync with Firestore

## Setup

1. Install Flutter and set up Android SDK
2. Create a Firebase project at https://console.firebase.google.com/
3. Add Android app to Firebase project and download google-services.json
4. Place google-services.json in android/app/
5. Run `flutter pub get`
6. Create an emulator: `flutter emulators --create emulator`
7. Run `flutter emulators --launch emulator`
8. Run `flutter run`
